# Definition of Ready (DoR) – CRM

Una historia se considera lista para desarrollo cuando:

- Tiene valor claro y usuario identificado.
- Contiene criterios de aceptación en formato Gherkin.
- Cuenta con wireframe enlazado en Figma.
- Tiene impacto arquitectónico identificado.
- Está estimada en Story Points.
- Dependencias identificadas.

---

# Definition of Done (DoD) – CRM

Una historia se considera terminada cuando:

- Está integrada en rama dev mediante Pull Request aprobado.
- Cumple todos los criterios de aceptación.
- Pruebas verificadas correctamente.
- Diagramas actualizados si hubo cambios.
- Figma actualizado si hubo cambios.
- Historia movida a Done en el tablero.
- Demo reproducible.
